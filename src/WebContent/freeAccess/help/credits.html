This library provides utilities to help localizing appliations built with many programming languages. This library is available under the BSD license. Information given here are covered by the CC by-nc-sa licence.

The interface is mainly built with the Dojo Toolkit. Dojo is an open source library supported by major companies. It is available under two licenses: BSD license and Academic Free License.

Some icons have been produced by Mark James, freely available to anyone under the Common Creative Attribution 2.5 license.
